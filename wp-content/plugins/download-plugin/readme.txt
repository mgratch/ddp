Download any plugin from your wordpress admin panel's Plugins page by just one click!

== Description ==

1. Just activate this plugin.
2. You can see <b>Download</b> link below each plugin name on plugins page.
3. Click on any of them and that plugin's zip will be downloaded to your computer.
4. Cheers!

= Download Theme =

If you want to download any theme from your wordpress admin panel's Appearance page, then use our other plugin - [Download Theme](https://wordpress.org/plugins/download-theme)

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Extract the zipped file and upload the folder `download-plugin` to `/wp-content/plugins/` directory.
